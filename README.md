# archlinux-cachyos-repository
Script de instalación de repositorios cachyos en arch linux y utilizar los paquetes. 

## Script de instalación 
Para instalar el script, se debe ejecutar el siguiente comando en la terminal:

Solo esta disponible para arch linux y solo verificado para las intel arc.

```bash
wget https://raw.githubusercontent.com/Mapachuelo/documentacion_linux/main/install.sh
chmod +x install.sh
./install.sh
```
Otra opción es 
```
git clone https://github.com/Mapachuelo/archlinux-cachyos-repository.git
cd archlinux-cachyos-repository
chmod +x install.sh 
./install.sh
```
